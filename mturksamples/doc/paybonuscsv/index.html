PayBonusCSV

PayBonusCSV is a utility that uses the Mechanical Turk API in Java to pay bonuses to a number of workers in batch.

PayBonusCSV is built on top of the Mechanical Turk sample package, and uses the Apache ANT utility to both compile and execute the java script.

File Setup

Before running the script, you must set up the csv input file and let the script know the secret key for your Mechanical Turk account.
The testinp.csv file is included in the samples directory. Each line contains a comma-separated list consisting of the assignment ID, the worker ID, the bonus amount and the reason for paying (in that order).

2C21QP6AUC26ERQ7ZEO3Y0FK0QG81X,A27ANNY9E0URA2,0.5,because
2C21QP6AUC26ERQ7ZEO3Y0FK0QG81X,A27ANNY9E0URA2,1,Good job!
2C21QP6AUC26ERQ7ZEO3Y0FK0QG81X,A27ANNY9E0URA2,2.22,Thank you.

This file pays three different bonuses to the same worker for the same assignment (but for different reasons). Notice that including commas (,) in the reason may lead to unexpected results. You may use a spreadsheet program such as excel to create your own csv file, and you may save it anywhere on your hard drive. The file mturk.properties contains the relevant keys to access your account. You can find your access keys by going to aws.amazon.com, hovering your mouse over "Your Web Services Account" in the top-right corner and selecting View Access Key Identifiers. Be sure to log-in with the same username and password you registered with your Mechanical Turk Requester account.

In the same file, choose the appropriate service_url for either the developer sandbox (to test things) or the production side (if you are confident that your system is correctly set-up).

Running

To run the script, open a terminal and navigate to the directory containing the build.xml file. Type ant at the command line, to verify that ant is installed:

This output demonstrates that ant is installed, and is referring to the correct build.xml file. To run the PayBonusCSV script, type ant paybonuscsv -Dbonuscsv=[location]

Make sure you include the entire path to the file, e.g.: bash-3.2$ ant paybonuscsv -Dbonuscsv="/Users/astorer/Work/eclipse/mturksamples/samples/testinp.csv"

As the script runs, it will echo back the information sent to the Mechanical Turk servers, and finally report a summary of the actions taken: [java] Awarded a total of $3.72 to 3 workers.
